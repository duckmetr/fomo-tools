# 🦊 Fomo Tools

**Fomo Tools** is a community tool for the game **Fomo Fighters**.  
It provides a searchable **database of all players** to make it easier to find opponents for **PvP battles**.

🌐 Live website:  
https://fomo-tools.netlify.app

---

# ⚔️ Features

### 🔎 Player Search

Quickly find any player in the Fomo Fighters ecosystem.

### 📊 Full Player Database

Browse the entire player list with **thousands of fighters** in one place.

### 🧬 Race Filtering

Filter fighters by **race** to find the best matchups for PvP.

### 📈 Level Filtering

Search players by **level range**.

### 💪 Power Sorting

Sort fighters by **Power (Low → High)** to analyze stronger opponents.

### 🏹 PvP Preparation

Use the database to:

- find targets for PvP battles
- analyze potential opponents
- explore the Fomo Fighters player base

### 📅 Database Updates

Player data is regularly updated to keep the list fresh and relevant.

# 🧠 Player Information

Each player card shows:

- 🧑 Player name
- 💪 Power
- 📏 Distance
- 🛡 Clan
- 🧬 Race
- 🎚 Level

This allows quick analysis before starting a PvP battle.

---

# 🖥 Interface

The interface is designed for **fast navigation and filtering**:

- sorting controls
- race filter
- level filter
- player cards grid
- total player counter

Example:

- **Total players tracked:** 23,000+
- **Sortable PvP targets**

---

# 🚀 Purpose

The goal of **Fomo Tools** is simple:

> Make PvP battles in **Fomo Fighters** easier by giving players a fast way to find opponents.

Instead of manually searching players, the tool allows **instant lookup and filtering**.
